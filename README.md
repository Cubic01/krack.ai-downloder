# Course Downloader (Krack.ai)

A Python script to automatically download videos and PDFs from a course on Krack.ai using authenticated cookies.
The script preserves the course structure by grouping lessons into their respective sections.

---

## Features

* Authenticated scraping using browser cookies
* Automatically fetches all lessons from a course
* Groups content by course sections (e.g., "Ukulele Introduction", "Happy Birthday")
* Downloads:

  * Video files (.mp4)
  * PDF resources
* Skips already downloaded files
* Clean folder structure

---

## Setup

### 1. Export Cookies

You need valid login cookies from your browser, get it Cookie-Editor Extension. (json Format)
